# Mini Game de Sobrevivência - Free Fire

Este é um projeto acadêmico em C que implementa um sistema de sobrevivência inspirado no jogo Free Fire, focando no estudo de estruturas de dados, algoritmos de busca e ordenação.

## Funcionalidades

1. **Sistema de Inventário**
   - Implementação usando estruturas estáticas (array)
   - Operações de inserção e remoção de itens
   - Gerenciamento de diferentes tipos de itens (Alimentos, Armas, Ferramentas)

2. **Algoritmos de Busca**
   - Busca Sequencial
   - Busca Binária (implementação iterativa)
   - Busca Binária (implementação recursiva)

3. **Algoritmos de Ordenação**
   - Selection Sort para ordenação do inventário

4. **Sistema de Torre de Fuga**
   - Verificação de recursos necessários
   - Construção da torre para escape

## Como Compilar

Para compilar o projeto, use o seguinte comando no terminal:

```bash
gcc main.c inventario.c busca.c ordenacao.c -o jogo
```

## Como Executar

Após compilar, execute o programa com:

```bash
./jogo
```

## Estrutura do Projeto

- `main.c` - Arquivo principal com o menu do jogo
- `inventario.h/c` - Implementação do sistema de inventário
- `busca.h/c` - Implementação dos algoritmos de busca
- `ordenacao.h/c` - Implementação dos algoritmos de ordenação

## Objetivos de Aprendizado

1. Compreensão de estruturas de dados em C
2. Implementação de diferentes tipos de busca
3. Análise e implementação de algoritmos de ordenação
4. Modularização e organização de código
